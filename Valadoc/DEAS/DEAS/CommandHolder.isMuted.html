public bool isMuted (int n)

Check if command of CommandInterface is muted

Short Description

Checks if command of CommandInterface instance, stored at the N-th element of CommandHolder is muted

Parameters:

n

number of element in array to mute the CommandInterface command

Returns:

true if the command is muted or doesn't exist,
false if the command is not muted
